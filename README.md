# asbf ( Asepit Brute Force )

# premfb 
# INSTALL

```
$ pkg update && pkg upgrade

$ pkg install python

$ pkg install git

$ pip install bs4

$ pip install requests

$ pip install cython

$ rm -rf asbf

```

# INSTALL REPOS

```

$ git clone https://github.com/privatescrip/asbf

```

# RUN

```

$ cd asbf

$ python premfb.py

```
